The Potato State Rivalry is the intrastate college football game in Idaho between the University of Idaho in Moscow and Idaho State University in Pocatello
This rivalry is over 100 years old with the first meeting in 1916 resulting in a 32-0 Vandal victory

Since Idaho rejoined the Big Sky conference the game has been played 8 times.

  1. October 18th, 2018 ISU 62 UI 28
  2. October 19th, 2019 UI 45 ISU 21
  3. April 3rd, 2021 ISU 24 UI 22
    • Played in spring covid season.
    • Last Bengal win in Pocatello.
    • Both head coaches fired by years end.
  4. November 20th, 2021 UI 14 ISU 0
  5. November 19th, 2022 #21 UI 38 ISU 7
  6. November 18th, #6 UI 63 ISU 21
  7. November 24th, #7 UI 40 ISU 17
  8. November 22nd, ISU 37 UI 16
